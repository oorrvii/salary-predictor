# 💰 Salary Predictor

A machine learning web app that predicts salary based on years of experience.

## 🔗 Live App
[Click here](https://salary-predictor-hnzc6ewucvq5wr6afl7itv.streamlit.app/)

## 📌 Project Overview
- Dataset: Salary vs Experience (30 samples)
- Model: Simple Linear Regression
- Accuracy: R2 Score 0.76, MAE $6,185

## 📊 Model Equation
Salary = 9332 × YearsExperience + 25609

## 🚀 Tech Stack
- Python, Scikit-learn, Streamlit
